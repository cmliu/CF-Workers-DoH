📖 功能说明

🔍 自动检测原理

本页面会自动使用各个 DoH 服务解析 www.google.com，通过分析返回的 IP 地址信息来判断 DNS 服务的纯净度。

检测流程：
1. 发送 DNS 查询请求到各个 DoH 服务
2. 获取解析结果的第一个 IP 地址
3. 查询该 IP 的归属信息（ISP、组织、AS号）
4. 检查是否包含 "Google" 相关标识

🟢 纯净状态

显示条件：返回的 IP 地址归属于 Google（ISP、组织或 AS 信息包含 "Google"）

意义：该 DoH 服务未被污染，返回真实的 Google 服务器地址

🔴 疑似污染

显示条件：返回的 IP 地址不属于 Google 服务器

可能原因：DNS 污染、地理位置限制、或运营商劫持

📊 显示信息说明

响应时间：从发送请求到收到响应的时间，越低越好

IP 地址：DoH 服务返回的解析结果

地理位置：显示格式为 "国家 地区 组织机构"

纯净度：基于 IP 归属判断的污染状态

💡 使用建议：
优先选择响应时间短且状态为"纯净"的 DoH 服务，以获得最佳的 DNS 解析体验。
